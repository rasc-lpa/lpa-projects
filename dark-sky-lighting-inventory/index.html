    Spectrum tools

    Screenshot from any of these apps and upload to the fixture's spectrum slot:

    Spectroid (Android, free) — real-time spectrogram
    Spectrometer+ (iOS) — camera-based light spectrum
    USB spectrometer software (Ocean Optics, Hamamatsu) — export as PNG/JPG
    Any SPD or CCT/CRI data screenshot from a calibrated meter
    Data & storage

    Data is stored in your browser's local storage. Use Export CSV regularly to back up your work. Photos and spectra are stored as base64 — roughly 0.5–2 MB per image.

    For multi-user or hosted use, see the GitHub repository for deployment options including Google Sheets integration and PWA packaging.
